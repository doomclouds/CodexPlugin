# Privacy

This local plugin does not transmit repository content, generated assets, screenshots, or logs to an external service by itself.

It reads and writes local files only when Codex invokes its skills or scripts. Any external network behavior would come from other tools the user explicitly asks Codex to use, not from this plugin.

Project assets should stay in the repository-local `docs/superpowers/` tree unless the user explicitly asks for a different destination.
